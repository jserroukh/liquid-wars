# Liquid Wars - Java Edition

Implémentation du jeu Liquid Wars en Java avec algorithmes de propagation sur grille et IA comportementale.

## Description

Le joueur contrôle une armée liquide qui se propage sur une grille selon des algorithmes de calcul de maillage. Le projet inclut un joueur artificiel capable de prendre des décisions de manière autonome, sans intervention humaine.

## Fonctionnalités

- Algorithmes de propagation sur grille
- Calcul de maillage
- IA comportementale — joueur autonome capable de décider et s'adapter
- Architecture orientée objet (CPOO)

## Prérequis

- Java 21
- Gradle 8+ (ou utiliser le wrapper Gradle inclus)

## Lancement
```bash
./gradlew build      # Compile le projet
./gradlew run        # Lance le jeu
```
